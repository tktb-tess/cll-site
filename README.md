# The Complete Lojban Language

I convert the Lojban grammar book "The Complete Lojban Language" into markdown, built it with Astro, and published it.

## License

Please see [LICENSE.md](./LICENSE.md).

## Run dev-server

```bash
git clone https://github.com/tktb-tess/cll-site.git
pnpm i --frozen-lockfile
pnpm run dev
```

## Generate HTML

```bash
pnpm run build
```

After executing above commands, it will be generated in the `./dist` directory.
